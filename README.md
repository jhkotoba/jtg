default extension  
java -jar jtg.jar /path/to/your/project
- `.java`
- `.yml`
- `.properties`
- `.css`
- `.scss`
- `.gradle`
- `.xml`
- `.js`
- `.html`


whitelist extension  
java -jar jtg.jar /path/to/your/project -w java,xml,html

blacklist extension  
java -jar jtg.jar /path/to/your/project -b log,class
